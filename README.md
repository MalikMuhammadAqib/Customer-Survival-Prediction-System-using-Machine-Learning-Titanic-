# 🚀 Customer Survival Prediction System | Machine Learning Case Study

## 🔹 Business Context
Organizations often need to predict **customer outcomes** based on historical behavior and demographics.  
In this case study, the Titanic dataset is used as a **simulated real-world customer survival problem**, where survival represents a positive outcome.

This project demonstrates my ability to:
- Translate a real-world problem into a machine learning solution
- Prepare and analyze structured data
- Build and evaluate predictive models
- Extract actionable insights from data

---

## 🔹 Problem Definition
**Objective:**  
Predict whether a customer (passenger) survives based on demographic and service-related features.

**Target Variable:**  
- `Survived` (0 = No, 1 = Yes)

---

## 🔹 Dataset Overview
| Feature | Description |
|------|------------|
| Pclass | Service level / customer tier |
| Sex | Gender |
| Age | Customer age |
| SibSp | Family relationships |
| Parch | Family relationships |
| Fare | Price paid |
| Embarked | Entry point |

---

## 🔹 Data Cleaning & Preparation
- Identified and handled missing values
- Encoded categorical variables
- Selected relevant predictive features
- Split data into training and testing sets

**Why this matters:**  
Real-world data is never clean. This step shows my ability to work with imperfect datasets.

---

## 🔹 Exploratory Data Analysis (EDA)
Key insights discovered:
- Gender significantly influences survival probability
- Higher service tiers show better survival outcomes
- Younger passengers had higher survival chances

Visualizations were used to validate these trends.

---

## 🔹 Model Development
Machine learning models applied:
- Logistic Regression

**Why Logistic Regression?**
- Interpretability
- Baseline classification performance
- Suitable for binary outcomes

---

## 🔹 Model Evaluation
Metrics used:
- Accuracy Score
- Confusion Matrix
- Classification Report

The model effectively identified survival patterns and provided interpretable results.

---

## 🔹 Key Business Insights
- Service level strongly impacts customer outcomes
- Female customers showed significantly higher survival rates
- Pricing (fare) correlates with survival probability

These insights could help organizations:
- Improve risk assessment
- Optimize customer segmentation
- Design targeted interventions
